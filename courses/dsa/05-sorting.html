Sorting algorithms (intro)

Bubble sort — O(n²)

Simple to understand: compare neighbors, swap if wrong order, repeat. Fine for learning, not for large data.

Built-in sort

nums = [5, 2, 8, 1]
nums.sort()
print(nums)

Python uses Timsort — efficient hybrid. In interviews you explain simple sorts; in projects you often use .sort() or sorted().
