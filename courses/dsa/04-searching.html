Searching algorithms

Linear search — O(n)

Check each element until found. Works on any list.

Binary search — O(log n)

Requires a sorted list. Compare middle, discard half, repeat.

def binary_search(nums, target):
    lo, hi = 0, len(nums) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1
