Lesson 9

Modules and organizing code

~35 min

Importing modules

Python ships with many libraries. You import them to reuse tested code.

import random
import math

print(random.randint(1, 10))
print(math.sqrt(16))

Your own modules

Put functions in helpers.py, then in main.py:

from helpers import greet

greet("OpenLearn")

Project structure tip

  • One clear entry file (e.g. main.py)
  • Helper logic in separate modules
  • Data files in a data/ folder
