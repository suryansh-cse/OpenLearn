Project 1 · Python

Personal Greeting Script

Congratulations — you've completed the foundations of Python: variables, data types, inputs, and operators. Now it's time to combine everything and build your first real mini project.

🎯 Project Goal

Build a Python program that asks the user for: their name, birth year, and favorite programming language. Then calculate their age and print a clean, personalized message.

📋 Requirements

Your program must do the following:

  • Ask the user for their name.
  • Ask for their birth year.
  • Ask for their favorite programming language.
  • Calculate the user's age using the year 2026.
  • Print a clean summary message.

✅ Example Output 

Hello Suryansh!
You are 19 years old.
Your favorite programming language is Python.
Keep learning and building 🚀

💡 Hints

  • Use input() to take user input.
  • Remember: input() returns a string.
  • Convert the birth year using int().
  • Use variables to store values.
  • You can print output using print() or f-strings.

name = input("Enter your name: ")

birth_year = int(
    input("Enter your birth year: ")
)

language = input(
    "Favorite programming language: "
)

age = 2026 - birth_year

print(f"Hello {name}!")
print(f"You are {age} years old.")
print(
f"Your favorite programming language is {language}."
)
print("Keep learning and building 🚀")
