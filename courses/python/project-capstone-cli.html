Capstone · Week 8

Build a CLI tool in Python

~3–5 hours

Goal

Combine everything: functions, files, argparse or a menu, and clear structure. Ship something you can demo.

Pick one idea

  • Study tracker — log hours per topic, show weekly summary
  • Expense logger — add expenses, export CSV, show totals
  • Quiz CLI — load questions from JSON, score the user
  • File organizer — sort files in a folder by extension

Must include

  • At least 2 modules OR 1 module + clear functions
  • Read or write a file
  • Handle bad input without crashing
  • README with how to run the project

Submission checklist

  1. Code runs from terminal with python main.py
  2. Sample data file included if needed
  3. You can explain each function in your own words
Build & verify

Capstone checklist

Ship your CLI tool, demo it once, then confirm each must-have item.
