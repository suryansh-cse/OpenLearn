Lesson 3

Conditionals (if / elif / else)

~30 min

Decisions in code

Programs branch based on conditions using if, elif, and else.

score = int(input("Score? "))

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
else:
    grade = "C"

print("Grade:", grade)

Comparison operators

== equal, != not equal, < > <= >=

Combining conditions

age = 20
has_id = True

if age >= 18 and has_id:
    print("Allowed")

Use and, or, and not to combine tests.
