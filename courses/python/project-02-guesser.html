Project 2 · Week 2

Number guessing game

~1 hour

Goal

The computer picks a random number. The player guesses until they get it right. Use conditionals, loops, and random.

Requirements

  • Secret number between 1 and 100
  • Print "too high" / "too low" hints
  • Count number of guesses

Hint

import random

secret = random.randint(1, 100)
guesses = 0

while True:
    guess = int(input("Your guess: "))
    guesses += 1
    if guess == secret:
        print(f"Correct in {guesses} tries!")
        break
    elif guess < secret:
        print("Too low")
    else:
        print("Too high")
