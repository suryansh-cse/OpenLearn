Project 3 · Week 3

Interactive CLI To-Do & Quiz App

~2 hours · Core structures & files

Milestone 3 Goal

Synthesize functions, collections (lists and dictionaries), string manipulation, and file I/O. You will build a multi-purpose command-line utility offering a menu-driven interface for: (1) managing a To-Do list saved to disk, (2) taking a mock programming quiz, and (3) generating secure random passwords.

Requirements

  • Main Menu Loop: A loop that displays options: 1. To-Do List, 2. Take Quiz, 3. Password Generator, 4. Exit.
  • To-Do Manager: Adds and lists tasks. Save the tasks to a text file (todo.txt) so they persist when the script is closed, and load them back on launch.
  • Quiz Taker: Loops over a dictionary of questions and answers. Checks user responses, shows correct options, and prints a final score (e.g. 2/3 correct).
  • Password Generator: Prompts for a password length and uses the standard library (random module) to assemble a secure random string of letters, numbers, and symbols.
