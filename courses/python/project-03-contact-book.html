Mini contact book

~1–2 hours

Goal

Store contacts in a list of dictionaries. Support add, list, and search by name.

Requirements

  • Menu loop: 1 Add, 2 List all, 3 Search, 4 Quit
  • Each contact: name, phone, email (email optional)
  • Search is case-insensitive partial match on name

Data shape

contacts = [
    {"name": "Ada", "phone": "555-0100", "email": "ada@example.com"},
]

Stretch

Save and load contacts from a JSON file using the json module.

Build & verify

Practice workspace

Build the contact book, run through each menu option once, then check off the requirements.
