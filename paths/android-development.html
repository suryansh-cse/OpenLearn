Android Development

Build Android applications with Kotlin and modern Android development practices

Intermediate 21 Lessons

Path Overview

This Android development path covers modern Android app development using Kotlin and the latest Android development tools. You'll learn to build native Android applications that work across millions of devices, from basic apps to complex applications with advanced features.

What You'll Learn

  • Kotlin programming for Android
  • Android app architecture (MVVM, Clean Architecture)
  • User interface design with Jetpack Compose
  • Data persistence and Room database
  • Networking with Retrofit and REST APIs
  • Google Play Store deployment
  • Material Design principles

Courses in This Path
