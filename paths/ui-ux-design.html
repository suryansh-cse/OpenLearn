UI/UX Design

Master user interface and user experience design principles and tools

Beginner 14 Lessons

Path Overview

This comprehensive UI/UX design path will teach you the fundamental principles of user-centered design, from understanding user needs to creating beautiful and functional interfaces. You'll learn industry-standard tools and methodologies used by professional designers.

What You'll Learn

  • User research and personas
  • Wireframing and prototyping
  • Visual design principles and typography
  • Usability testing and user feedback
  • Design systems and component libraries
  • Industry-standard design tools (Figma, Adobe XD)
